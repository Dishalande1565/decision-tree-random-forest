# Decision Tree & Random Forest — Task 10

## AI & ML Internship

### About
Classification using Decision Tree and Random Forest
on Breast Cancer Dataset.

### Dataset
- Name    : Breast Cancer Dataset
- Rows    : 569
- Features: 30 numeric features
- Classes : Malignant / Benign

### Steps Completed
1. Preprocessed Data
2. Train-Test Split 80/20
3. Decision Tree Training and Visualization
4. Random Forest Training (100 trees)
5. Evaluated Accuracy and F1 Score
6. Model Comparison
7. Feature Importance Plot

### Results
- Decision Tree Accuracy : 93-95%
- Random Forest Accuracy : 96-98%
- Random Forest performs better ✅

### Tools Used
- Python 3, Jupyter Lab
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

### How to Run
1. Clone this repository
2. Open Jupyter Lab
3. Open decision_tree_random_forest_task10.ipynb
4. Run all cells → Shift + Enter
